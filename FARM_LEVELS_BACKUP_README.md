# 🔒 농장 레벨 초기값 백업

이 디렉토리에는 HappyTree 게임의 **초기 농장 레벨 설정값**이 백업되어 있습니다.

## 📁 백업 파일

### 1. TypeScript 백업
- **파일**: `src/lib/farm-levels-initial.ts`
- **용도**: 코드에서 직접 import 가능
- **함수**:
  - `getInitialLevels(farmId)`: 특정 농장의 초기값 반환
  - `getAllInitialLevels()`: 모든 농장의 초기값 반환

### 2. JSON 백업
- **파일**: `INITIAL_FARM_LEVELS.json`
- **용도**: 외부 도구나 시스템에서 참조 가능
- **형식**: 표준 JSON 형식

## 🔧 복구 방법

### TypeScript에서 복구
\`\`\`typescript
import { getInitialLevels, getAllInitialLevels } from './lib/farm-levels-initial';

// 특정 농장 복구
const farm1Initial = getInitialLevels(1);

// 모든 농장 복구
const allInitial = getAllInitialLevels();

// farm-calculator.ts에 적용
import { FARM_LEVELS } from './lib/farm-calculator';
FARM_LEVELS[1] = getInitialLevels(1);
\`\`\`

### JSON에서 복구
\`\`\`javascript
const fs = require('fs');
const initialData = JSON.parse(fs.readFileSync('INITIAL_FARM_LEVELS.json', 'utf8'));
console.log(initialData.farms['1']); // 농장 1 데이터
\`\`\`
